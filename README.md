# Employee Management System

A console-based Employee Management System developed using Java.

## Features

* Add Employee
* View Employees
* Search Employee
* Delete Employee

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* Scanner Class

## Project Structure

src/
├── Employee.java
└── Main.java

## How to Run

1. Open the project in IntelliJ IDEA.
2. Run Main.java.
3. Use the menu options to manage employees.

## Version

Version 1.0
